EventBuilder

Description

This project was a project I made with Spigot API using Java of course. I tried utilizing Java's functional paradigm in this project.

Goals

The goals here was to create a framework on top of the Spigot Event API. Spigot Event API uses reflection which can be kinda slow, so an improvment would be to just use an event executor which delegates a given consumer of an event type. The framework also embraces the builder pattern.

Result

The result was good. When I tested the finished product it was clearly faster than the normal Spigot Event API.
