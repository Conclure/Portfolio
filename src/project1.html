DeluxeAsyncJoinLeaveMessage

Description

This project was a project I made with Spigot API using Java of course. While some people argue that concurrent execution is optimized the catch is that it is not always. Synchronous execution can be very fast also.

Goals

The goal was to by intention create a simple join leave plugin which would broadcast join and leave messages whenever a player would join or leave the server. Another intention here was to use anti patterns as well as bad practices.

Result

Whilst the result was not really great I did get some attention for it at least which was fun.
